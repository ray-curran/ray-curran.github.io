SQL is a programming language designed to store data in relational tables. These tables can be linked to each other to for one-to-one, many-to-one, and many-to-many relationships. This cheat sheet displays commands to use when creating SQL queries. In these examples, I will use "table" to refer to a general table title and "column#" to refer to specific columns.


SELECT ALL DATA FROM A TABLE

The asterisk wildcard in this example tells the system to select all data and all columns from the table listed:

SELECT * FROM table;

ORDER DATA BY A SPECIFIED COLUMN

To order data by a specific column in ascending order:

SELECT * FROM table ORDER BY column1;

To order data by a specific column in descending order:

SELECT * FROM table ORDER BY column1 DESC;

SELECT DATA THAT MEETS CERTAIN CRITERIA

Find records in a table that meet criteria based on a specific column:

SELECT * FROM table WHERE column1="searchtext";

You can also use the % wildcard within a text search to find data that contains the text you're searching for:

SELECT * FROM table WHERE column1 LIKE "%search text%";

SELECT SPECIFIC COLUMNS FROM A TABLE

List columns instead of using the * wildcard to see only specific data from a table"

SELECT column1, column2 FROM table;

GROUP SELECTED DATA IN A TABLE

Group data in your output by matching on a specific column:

SELECT * FROM table GROUP BY column1;

DISPLAY DATA FROM TWO JOINED TABLES

Display all data from two joined tables:

SELECT * FROM table1 INNER JOIN table2 ON table1.column1=table2.column4;

Display specific data from two joined tables

SELECT table1.column2, table1.column3, table2.column4 FROM table1 INNER JOIN table2 ON table1.column1=table2.column4;

FIND DISTINCT DATA FROM TABLE

Display only distinct data in a column from a table:

SELECT DISTINCT column1 FROM table;

UPDATE DATA IN A TABLE WHERE CERTAIN CONDITIONS ARE MET

Specify what columns to change and under what conditions:

UPDATE table SET column1='replacement text' WHERE column2='search text';

DELETE RECORDS IN TABLE THAT MEET CRITERIA

List the criteria and the table to delete records from:

DELETE FROM table WHERE column1="search text";
